Methods & where they're defined

FiniteAlgebra
    name - string name of algebra
    description - string description of algebra

SingleElementSetAlgebra( FiniteAlgebra ) - algebra with single set of elements
    about - prints info about the algebra
    deepcopy
    dump - saves algebra to file in JSON format
    dumps - outputs algebra as JSON string
    elements - list of names of the algebra's elements (strings)
    has_identity - returns True if the algebra has an identity
    has_inverses - returns True if the algebra has inverses
    identity - returns the algebra's identity element, if it exists
    inv - returns and element's inverse element
    is_abelian - returns True if the algebra is abelian
    is_associative - returns True if the algebra is associative
    is_commutative - same as is_abelian
    order - returns the order of the algebra
    table - returns the algebra's Cayley table
    to_dict - outputs the algebra as a Python dictionary

Magma( SingleElementSetAlgebra ) - a set with a binary operation
    center - elements that commute with every element of the algebra
    center_algebra - subalgebra generated by the center
    closure - smallest, closed set of elements for a given set of elements
    closed_subsets_of_elements - all unique, closed, proper subsets of elements
    direct_product - [defined using the Python __mult__ method]
    is_cyclic
    is_division_algebra
    isomorphic
    op
    power - the direct product of this algebra with itself, n times
    proper_subalgebras
    reorder_elements - returns new group made from self with elements reordered
    generators - list of elements that, individually, generate the entire algebra
    subalgebra_from_elements
